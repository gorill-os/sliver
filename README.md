## How to use

### Create Box

If you use distrobox:

    distrobox create -i ghcr.io/gorill-os/sliver -n sliver --hostname SLIVER
    distrobox enter sliver
    
If you use toolbx:

    toolbox create -i ghcr.io/gorill-os/sliver -c sliver --hostname SLIVER
    toolbox enter sliver
